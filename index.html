Help

Lưu Ý:

Hiện chỉ hỗ trợ thoại, lựa chọn, vũ khí, vật phẩm, kỹ năng, kẻ thù

Cách dùng :

Chọn tệp bản đồ, các map được đặt trong thư mục Data của dự án vd Map001.json, v.v.

Hãy đợi trong quá trình tải file, việc tải file sẽ phụ thuộc vào kích thước file và máy của bạn.

Ở bên trái là các Event của map, kèm theo id của event và trang, event và trang sẽ không hiện nếu nó không có text.

Chọn một event và 1 khung chứa văn bản sẽ xuất hiện.

Tại đây sẽ chứa text và khung để sửa đổi.

Để xác định thay đổi nhấn bên ngoài khung hoặc enter

Khi thay đổi xong nó sẽ chuyển sang màu xanh lá.

Với vật phẩm/kỹ năng:

Chỉ cần mở file cần và thay những gì muốn thay.

Nhấn nút "Tải xuống" sẽ tải xuống file đã được sửa.

Nếu không tải xuống thì các thay đổi sẽ mất khi mở lại.

Nếu có lỗi phát sinh hãy truy cập
